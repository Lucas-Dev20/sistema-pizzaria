=== Sistema de Gerenciamento de Pizzaria ===

Requisitos:
- Java 17
- JavaFX 17
- MySQL 8.0
- IntelliJ IDEA (ou outra IDE)

Diagrama de Classes se encontra em: src/main/resources/br/edu/ufersa/pizzaria/images/dc_final.jpg

Passos para executar:

1. Clone o projeto do GitHub.

2. Abra o MySQL.

3. Execute o arquivo: src/main/resources/br/edu/ufersa/pizzaria/schema.sql
   schema.sql

4. Altere, se necessário, o usuário e senha do banco na classe:
   ConnectionFactory.java

OBS: Usamos o terminal para verificar e acessar todos os dados, colunas e tabelas.
Se for usar o terminal basta executar o codigo para ser criado o banco de dados.

Se for usar o MYSQL WORKBENCH faça os seguintes passos:
1 - Abra o MySQL Workbench.
2 - Conecte-se ao seu servidor MySQL.
3 - Clique em File → Open SQL Script.
4 - Selecione o arquivo schema.sql.
Por fim o conteúdo será aberto em uma nova aba.

6. Abra o projeto na IDE.

7. Execute a classe Main.java.
